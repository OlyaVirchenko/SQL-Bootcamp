# Проект: SQL Bootcamp (School21)
 
**Технологии:** SQL, PostgreSQL, DataGRIP (JetBrains)  

## О проекте

SQL Bootcamp в School21 — интенсивный курс по изучению языка SQL и работе с базами данных. Программа построена на ежедневной практике: участники выполняют упражнения и презентуют решения перед группой.


Цель проекта — отработать практические навыки написания SQL‑запросов для решения реальных задач анализа и обработки данных.

## Ключевые навыки и технологии

* **SQL:** SELECT, JOIN, UNION, VIEW, транзакции, функции PostgreSQL.
* **PostgreSQL:** работа с индексами, последовательностями, уровнями изоляции.
* **Data Governance:** политики управления данными.
* **DWH и ETL:** основы работы с хранилищами данных и процессами извлечения‑преобразования‑загрузки.
* **Инструменты:** PostgreSQL (СУБД), DataGRIP от JetBrains (IDE).

## Программа и выполненные задачи

### Дни 0–3: основы SQL
* Отработка базового синтаксиса SQL.
* Написание запросов с использованием `SELECT`, `JOIN`, `UNION`.
* Извлечение и комбинирование данных из нескольких таблиц.

### День 4: виртуальные представления и снимки данных
* Работа с виртуальными представлениями (`VIEW`).
* Создание физических снимков данных для оптимизации запросов.
* Оптимизация доступа к данным через представления.

### Дни 5–7: управление данными и структура БД
* Изучение политик управления данными (Data Governance).
* Реализация индексов БД для ускорения запросов.
* Работа с последовательностями БД (sequences) для автогенерации значений.

### День 8: транзакции и уровни изоляции
* Освоение транзакций и их свойств (ACID).
* Анализ уровней изоляции и их влияния на параллельность и консистентность данных.
* Решение задач по обеспечению целостности данных.

### День 9: функции PostgreSQL
* Создание пользовательских функций для обработки данных.
* Программирование логики обработки внутри БД.
* Применение функций для решения комплексных задач анализа.

## Практический проект: Team01 (DWH + ETL)

Работа с хранилищем данных (DWH) и процессами ETL, включая обработку данных с аномалиями.
